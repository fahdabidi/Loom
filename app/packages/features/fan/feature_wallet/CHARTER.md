# feature_wallet Charter

## Scope

Fan wallet, simulated purchases, memberships, no-ad premium, entitlements, and allocation statements.

## Allowed contracts

- Fan Wallet API
- Entitlement Ledger API
- Receipt Ledger API
- Settlement Engine API

## Owned screens

Defined in Phase 6.

## Forbidden imports

Other features, `loom_fake_backend`, `loom_local_store`, and `loom_seed_data`.
