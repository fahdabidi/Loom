# feature_playback Charter

## Scope

Video/post playback, playback authorization, ad-supported playback, and completion receipts.

## Allowed contracts

- Playback Authorization API
- Content Host API
- Receipt Ledger API
- Creator Metadata API

## Owned screens

- PlaybackScreen

## Forbidden imports

Other features, `loom_fake_backend`, `loom_local_store`, and `loom_seed_data`.
