# Loom Demo App Workspace

This is the Flutter melos workspace for the Creator Studio + Fan App demo.

## Phase 0 working-tree decision

The workspace lives in-repo at `/mnt/c/Users/fahd_/OneDrive/Documents/Loom/app` so the generated app and packages remain versioned with the existing planning docs. This is slower than a native WSL filesystem, but it keeps the implementation colocated with the repository the user opened.

All app workflow commands should be run from WSL Ubuntu, not PowerShell.
